# 🚀 AI Resume Analyzer

An AI-powered web application that analyzes resumes, provides scores and feedback, matches keywords with job descriptions, and offers real-time improvement suggestions via an AI chatbot.

---

## ✨ Features

- 📄 **PDF Upload** — Drag & drop or click to upload your resume
- 🤖 **AI Analysis** — Get a resume score and personalized feedback
- 🔍 **Keyword Matching** — Compare your resume with any job description
- 💬 **AI Chatbot** — Ask questions and get real-time resume improvement suggestions

---

## 🛠️ Tech Stack

- **React** — Frontend framework
- **Vite** — Build tool
- **Tailwind CSS** — Styling
- **react-pdftotext** — PDF text extraction
- **Gemini API** — AI analysis and chatbot

---

## 🚀 How to Run

# 1. Clone the repo
git clone https://github.com/username/resume-analyzer.git

# 2. Enter folder
cd resume-analyzer

# 3. Install dependencies
npm install

# 4. Add your API key — create .env file
VITE_GEMINI_API_KEY=your-key-here

# 5. Run the project
npm run dev
